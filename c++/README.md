A growing collection of **C++ programs** covering core programming concepts, data structures, algorithms, and problem-solving practice — organized for quick reference and learning.

---

## 📂 Repository Structure

```
c++/
├── Graph/                 # Graph algorithms (BFS, DFS, etc.)
├── stl/                   # Standard Template Library practice
├── Dp & sell.cpp          # Dynamic Programming - buy/sell problems
├── array.cpp              # Array operations
├── binarySearch.cpp       # Binary search implementation
├── demo.cpp / demo.exe    # Demo / scratch program
├── function.cpp / .exe    # Functions in C++
├── mergeSorting.cpp / .exe# Merge sort algorithm
├── quickSorting.cpp       # Quick sort algorithm
├── sorting.cpp / .exe     # General sorting examples
├── oops.cpp / .exe        # OOP concepts (classes, inheritance, etc.)
├── input.txt / output.txt # Sample I/O files for testing
└── README.md
```

## ⚙️ How to Compile & Run

```bash
g++ filename.cpp -o output
./output
```

Example:
```bash
g++ mergeSorting.cpp -o mergeSorting
./mergeSorting
```

---

## 📌 Notes
- `.exe` files are precompiled outputs (Windows) — feel free to recompile on your system.
- `input.txt` / `output.txt` are used for file-handling based programs.
